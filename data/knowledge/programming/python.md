# Python

Python is a programming language that is meant to be beginner-friendly while also being deep enough for experts and professional developers.

## Key Features

- Simple syntax
- No `{}` of `;`
- Lots of frameworks for different purposes

## Syntax

To print text into the console, you write `print(text)`. For example, 
```python
print('Hello World!')
```
prints:
```text
Hello World!
```
To create variables, you write `var = 'Hello World!'`. Combining both results in this code:
```python
var = 'Hello World'
print(var)
```
which also prints:
```text
Hello World!
```

But you can also change the variable:
```python
var = "Hello World!"
print(var)
var = "Hello!"
print(var)
```
prints:
```text
Hello World!
Hello!
```

Even though the `print()` didn't change, the output changed because of the variable.
